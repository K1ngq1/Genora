# 节点右键、打组与框选设计

## 目标

为无限画布增加节点右键菜单、复制剪切粘贴、真实容器打组、组复制内部连线、整组展开连接，以及左键框选和中键平移。

## 右键菜单

- 右键节点打开节点操作菜单。
- 菜单包含复制、剪切、粘贴、打组、取消打组和删除。
- 单节点右键时操作该节点；已有多选且右键节点属于选区时，操作整个选区。
- 粘贴使用应用内部剪贴板，在右键位置附近创建副本。
- 支持 `Ctrl/Cmd+C`、`Ctrl/Cmd+X`、`Ctrl/Cmd+V`。

## 复制与剪切

- 复制普通节点时复制节点数据，不复制运行中的 `busy` 状态。
- 复制真实组时复制组、所有直接成员及成员之间的内部连线。
- 所有新节点和新连线生成新 ID。
- 粘贴后整体偏移，避免覆盖原节点。
- 剪切先写入内部剪贴板，再删除原节点。

## 真实容器组

- 选择至少两个顶层普通节点后可以打组。
- 组使用 React Flow `parentId` 建立真实父子关系。
- 组节点显示标题、边框及左右连接端口。
- 拖动组会带动全部成员。
- 成员限制在组边界内。
- 取消打组后成员保留当前绝对位置和已有连线。
- 首期不允许嵌套组。

## 组连接

- 组本身不参与生成。
- 从组端口连接到已有节点或连接到新节点时，不保留组边。
- 系统展开组的直接成员，为每个成员创建到目标节点的独立连线。
- 组建立连接后再新增成员，不自动补充历史连线。

## 框选与平移

- 空白画布左键拖动创建框选区域。
- 框选区域与节点部分相交即可选中节点。
- `Shift + 左键框选` 追加或切换选区。
- 鼠标中键按住拖动画布。
- 左键拖动节点仍用于移动节点或整组。
- 框选命中组边框时选择整个组；框选组内成员时选择具体成员。

## 保存恢复

- 组节点、`parentId`、成员相对位置、尺寸和内部连线随项目保存。
- 现有保存结构继续序列化 React Flow 节点，不建立第二套分组数据。

## 限制

- 不支持嵌套组。
- 不使用系统剪贴板，不读取其他应用复制内容。
- 不自动重连组建立连接之后新增的成员。
